# Interaction Interfaces
## CLI (`adk run <filename>`)
    :-> CLI Command directly runs agents in the command line. 
## Web UI (`adk web`)
    :-> Spins up a brand new Angular UI for interact with agent. And this also has multimodel capabilities, where you can interact with the agent using voice or voice.
## API Server (`adk api_server`)
    :-> Exposes agent as a REST endpoint.
## API (`Python`)
    :-> A programmatic way to call agent.
